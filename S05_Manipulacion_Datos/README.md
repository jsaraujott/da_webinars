En estos ejercicios se busca practicar los siguientes conceptos básicos de programación en **Python**:

* Manejo de variables
* Manejo de cadenas de texto
* Manejo de listas
* Manejo de diccionarios
* Definición y uso de funciones
* Uso de bucles
* Uso de condicionales

Posterior a ello, en el caso de estudio y ejercicios adicionales se busca prácticar los fundamentos de manipulación de datos con **Pandas**. Adicionalmente, se va a introducir los primeros criterios para un adecuado ordenamiento de proyectos de análisis de datos:

* Entendimiento del contexto
* Entendimiento de los datos
* Preparación de los datos
* Análisis de datos

De manera complementaria, se pretende dar una primera aproximación hacia la visualización de resultados mediante gráficos con **Matplotlib**.
